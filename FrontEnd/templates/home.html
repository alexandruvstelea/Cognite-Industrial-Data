Welcome to Cognite Industrial Data Tracker

This project aims to collect and process data provided by the Open Industrial Project for learning purposes.It uses the API provided by the open industrial platform to extract the data and stores it in a relational database. The database is refreshed every 8 days so that it can always have the latest data provided. It is then provided to the front end of the app for viewing using a REST API.
